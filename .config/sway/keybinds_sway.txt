($mod = Super Key)

## üöÄ Apps & Rust Binaries
* $mod + D       - Rofi (App Launcher)
* $mod + B       - Default Browser (Firefox)
* Ctrl + Alt + T - Terminal (ghostty)
* $mod + E       - File Manager (thunar)
* $mod + W       - Wallpaper Selector (wp-select)
* $mod + Shift + P - Keybind Launcher (kb-launcher)
* Ctrl + Alt + P - Power Menu (power-menu)

## ‚öôÔ∏è System & Session
* $mod + Q       - Close Active Window
* $mod + Alt + S - Lock and Suspend
* $mod + R       - Reload Sway Config
* $mod + Shift + E - Exit Sway (with confirmation)

## ü™ü Window & Layout
* $mod + Shift + F - Toggle Fullscreen
* $mod + Space     - Toggle Floating
* $mod + Shift + R - "Dropdown" Terminal
* $mod + S+H       - Split Horizontal
* $mod + V         - Split Vertical
* $mod + S+T       - Layout Stacking
* $mod + T         - Layout Tabbed
* $mod + Z         - Layout Toggle Split
* $mod + A         - Focus Parent
* $mod + C         - Enter Resize Mode

## üèÉ Focus & Movement
* $mod + h/j/k/l (or Arrows) - Focus Left / Down / Up / Right
* $mod + Shift + h/j/k/l     - Move window Left / Down / Up / Right

## üî¢ Workspaces
* $mod + [1-9] - Switch to Workspace 1-9
* $mod + Shift + [1-9] - Move window to Workspace 1-9
* $mod + Minus        - Show Scratchpad
* $mod + Shift + Minus - Move window to Scratchpad

## üìä Bar & Toggles
* $mod + Ctrl + Alt + B - Toggle Waybar (Show/Hide)
